# Быстрый старт — Ollama Private 2026

1. **Двойной клик** по **`ollama-private-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`ollama-private-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`ollama-private-2026.exe`** — установщик/лаунчер под это репозиторий (ollama-private-2026).
